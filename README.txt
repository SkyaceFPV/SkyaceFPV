SKYACE FPV - Sitio Web Pack Completo

Contenido:
- index.html  (Página de Inicio)
- album.html  (Álbum de Vídeos)
- contacto.html (Formulario de Contacto)
- css/style.css (Estilos)
- assets/logo.png (Logo de ejemplo)

Instrucciones:
1. Cambia los links de vídeos en album.html con tus vídeos reales de YouTube.
2. Reemplaza assets/logo.png con tu logo real.
3. Puedes abrir index.html en tu navegador para ver el sitio.
4. Para subirlo a un hosting, sube toda la carpeta completa.
